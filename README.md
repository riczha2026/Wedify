# 💍 Wedify

AI-powered wedding planning application (Project Management Initiative)

## 📌 Overview
Wedify is a hypothetical project developed as part of the UCSB PACE Intro to Project Management course. It focuses on planning and designing an AI-powered mobile app that simplifies wedding planning through personalized recommendations and a swipe-based interface.

## 🚀 Features
- AI-driven vendor recommendations  
- Swipe-based matching interface  
- Personalized planning (budget, style, preferences)  
- Timeline and coordination tools  
- Privacy-focused user data handling  

## 🎯 Objectives
- Build a Minimum Viable Product (MVP) in 6 months  
- Deliver an enhanced version within 8 months  
- Achieve 75%+ user satisfaction during testing  

## 🧠 Scope
**In Scope**
- Mobile app design (UI/UX)  
- Frontend & backend development  
- AI recommendation system  
- User authentication & database  

**Out of Scope**
- Payment processing  
- Vendor dashboards  
- Web version (initial phase)  

## 🛠️ Tech Stack (Planned)
- Frontend: React Native / Flutter (TBD)  
- Backend: Node.js / Firebase  
- AI: GPT / Claude APIs  
- Cloud: AWS
